# Infos


Get the information about the API in the current country 
## Examples
[https://api.deezer.com/infos/](https://developers.deezer.com/api/explorer?url=infos/ "https://api.deezer.com/infos/")
## Fields
Name | Description | Type
--- | --- | ---
country_iso | The current country ISO code | string
country | The current country name | string
open | Indicates if Deezer is open in the current country or not | boolean
offers | An array of available offers in the current country | array
