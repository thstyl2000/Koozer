# Kodi artwork placeholders

Binary artwork files are not stored in this repository. Download the assets using the links below and save them to the indicated paths.

## Icon (512x512 PNG)
- Download: https://placehold.co/512x512/png?text=Koozer+Icon
- Save as: `resources/media/icon.png`
- Kodi spec: 512x512 PNG icon.

## Fanart (1280x720 JPG)
- Download: https://placehold.co/1280x720/jpg?text=Koozer+Fanart
- Save as: `resources/media/fanart.jpg`
- Kodi spec: 1280x720 JPG fanart.

After downloading, verify the dimensions match the Kodi artwork requirements before packaging the addon.
